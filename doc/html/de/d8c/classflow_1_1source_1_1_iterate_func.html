Detailed Description

template<typename IteratingFunction, typename T, std::size_t N>
class flow::source::IterateFunc< IteratingFunction, T, N >

Stream source that generates an infinite stream using successive calls to an iterated function. Initial values are given, then later values are calculated using previous values in the stream.
