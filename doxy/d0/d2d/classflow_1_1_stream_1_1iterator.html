Detailed Description

template<typename Source>
class flow::Stream< Source >::iterator

Allows STL-like iterator access to the stream.

Stream iterators are forward and single-pass, once a stream has been iterated in any way (including terminal operations), the iterators are invalidated.

Additionally, only the first iterator ever used is valid. To extend the lifetime of a stream, use one of the to_container terminal operations.
