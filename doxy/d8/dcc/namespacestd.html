Detailed Description

Overload operator<< for std::tuple and std::pair.

Defines operator<< for std::tuple and std::pair. This definition is required by dump() for a stream containing these types, such as a stream that uses the default zip() operation. For example,

intstream | zip(more_ints) | dump(); // requires std::operator<< for tuples

The std namespace can be removed by defining #define FLOW_PRINTTUPLE_NO_STD before including flow.h.

// remove std::operator<< for tuples and pairs
#define FLOW_PRINTTUPLE_NO_STD
#include <flow.h>

This will cause statements like the above to be a compile error unless the operator is defined elsewhere inside of the std namespace. If FLOW_PRINTTUPLE_NO_STD is defined, then operator<< is defined inside of the flow namespace instead.

Overload operator<< to display pair types.

The pair is printed as (first, second). Each item is displayed using operator<<.

This is defined in the std namespace to allow dump() to work correctly. To exclude this function from being defined in the std namespace, #define FLOW_PRINTTUPLE_NO_STD will remove it and define it in the flow namespace instead.
