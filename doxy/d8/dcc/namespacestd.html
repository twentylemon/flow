Overload operator<< to display tuple types. The tuple is printed as

(item1, item2, ...)

. Each item is displayed using operator<< on the elements.

This is defined in the std namespace to allow dump() to work correctly. To exclude this function from being defined in std, #define FLOW_PRINTTUPLE_NO_STD will remove it.
