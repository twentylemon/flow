flow::uncurry
auto uncurry(CurryFunction curry_function)
Uncurries the specified function. Given a function which takes several parameters, converts that function into a function which takes one std::tuple of those arguments instead. This is useful when zipping streams together as it allows you to write a function which takes normal arguments instead of a tuple. For example, this allows the syntax: intlist | zip(more_ints) | map(uncurry([](int l, int r){ i and j things })) | ... as opposed to using std::tuple and std::get.
