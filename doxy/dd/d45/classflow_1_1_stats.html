Detailed Description

template<typename T, typename ResultType, bool MinMax, bool MedianMode>
class flow::Stats< T, ResultType, MinMax, MedianMode >

Class to wrap all the statistics returned by stats().

The count, mean, standard deviation, variance, sum and sum of squares of the stream elements are always calculated. Conditionally calculates additional statistics like median and mode, etc. If those values are not calculated, using the getter methods will result in a compiler error.
