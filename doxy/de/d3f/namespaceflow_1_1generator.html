template<typename IteratingFunction , typename... Args>
auto flow::generator::iterate ( IteratingFunction  function,
Args &&...  initial 
)

Creates an infinite Stream where values are created using successive calls to the function , starting with the values given. For example, iterate(std::plus<int>(), 0, 1) would result in the stream 0, 1, 1, 2, 3, 5, 8.... 0 and 1 are given, then the rest are calculated as 0+1=1, 1+1=2, f_i = f_{i-2} + f_{i-1}.

Parameters
functionThe iterating function.
...initialThe values the stream will contain, and initial values to send to the iterating function.
Returns
An infinite stream that iterates the function given.
