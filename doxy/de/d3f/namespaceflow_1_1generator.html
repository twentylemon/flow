Detailed Description

Factory methods to create stream objects, all of which are also aliased in the flow namespace.

The functions in the flow::generator namespace all produce Stream objects.

Also defines operator| for container types to be a shorthand for from(). A container is defined to be any class with a const_iterator inner class, and methods begin() and end().

auto flow::generator::closed_range ( T &&  lower,
T &&  upper 
)

Creates a stream containing the elements in the range [lower, upper], including upper .

The stream elements are incremented using operator++, and the range is computed using operator<=. The stream ends once element <= upper returns false. That is, the stream produced is equivalent to:

iota(lower) | take_while([upper](T e) { return e <= upper; })

Parameters
lowerThe lower bound of the range of elements to appear in the stream, inclusive.
upperThe upper bound of the range of elements to appear in the stream, inclusive.
Returns
A stream containing the elements in the range [lower, upper].

Creates a stream containing the elements in the range [lower, upper], including upper counting up by increment each step.

The stream elements are incremented using the function operator+(T, U) with the increment on the right hand side of the operand, and the range is computed using operator<=. The stream ends once element <= upper returns false. That is, the stream produced is equivalent to:

iota(lower, increment) | take_while([upper](T e) { return e <= upper; })

Parameters
lowerThe lower bound of the range of elements to appear in the stream, inclusive.
upperThe upper bound of the range of elements to appear in the stream, inclusive.
incrementThe value by which to increment each step in the stream.
Returns
A stream containing the elements in the range [lower, upper] incremented using operator+ increment .

template<typename IteratingFunction , typename... Args>
auto flow::generator::iterate ( IteratingFunction  function,
Args &&...  initial 
)

Creates an infinite stream where values are created using successive calls to the function , starting with the values given.

The next stream element is computed using the previous n stream elements applied to function , where n is the number of parameters that function takes.

For example, iterate(std::plus<int>(), 0, 1) would result in the stream 0, 1, 1, 2, 3, 5, 8... (the Fibonacci numbers). 0 and 1 are given, then the rest are calculated as 0+1=1, 1+1=2, 1+2=3, and f_i = f_{i-2} + f_{i-1} in general.

Parameters
functionThe iterating function.
...initialThe first values the stream will contain, and initial values to send to the iterating function to calculate further elements.
Returns
An infinite stream that iterates the function given.
