drop_while.cpp

A simple flow::drop_while example. This operation removes elements from the front of the stream as long as the elements return true for a given predicate. Once an element returns false, all elements after and including that element are kept, even if later elements would be dropped by the predicate.

See also
filter(), take_while()
#include <examples.h>
void drop_while_example() {
from({ 1, 2, 3, 4 }) | drop_while([](int i) { return i < 2; }) | dump(); // 2 3 4
from({ 1, 2, 3, 4 }) | drop_while([](int i) { return i == 0; }) | dump(); // 1 2 3 4
from({ 1, 2, 1, 1 }) | drop_while([](int i) { return i == 1; }) | dump(); // 2 1 1
from({ 1, 2, 3, 4 }) | drop_while([](int i) { return i > 0; }) | dump(); // empty
}
